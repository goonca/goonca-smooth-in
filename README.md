# Goonca Smooth-In

Smooth-In containers while scrolling

![Screenshot](goonca-smooth-in.gif)

## Getting started
```
$ npm install goonca-smooth-in
```
add goonca-smooth-in.js into your page
```
<script src="goonca-smooth-in" type="text/javascript"></script>
```
